## SEO Content

> ~606 tokens

### Keyword Research Workflow

1. Analyze Search Console data for existing opportunities
2. Identify low-competition, high-intent keywords
3. Group keywords by topic cluster
4. Classify search intent (informational vs. transactional vs. navigational)
5. Prioritize by search volume, difficulty, and business relevance

### On-Page SEO Checklist

- [ ] Title tag includes target keyword (under 60 characters)
- [ ] Meta description includes keyword (under 155 characters)
- [ ] H1 contains target keyword
- [ ] Target keyword in first paragraph
- [ ] Related keywords used throughout (semantic SEO)
- [ ] Clear H2/H3 hierarchy addressing subtopics
- [ ] Short paragraphs (2-3 sentences)
- [ ] Internal links to related pages
- [ ] FAQ section for long-tail keywords
- [ ] Content length appropriate (1000+ words for blog posts)

### Blog Post Structure

```
Title: <keyword-rich, under 60 chars>
Meta: <compelling description, under 155 chars>

# H1: <title with keyword>

<Introduction with hook and keyword in first paragraph>

## H2: <subtopic addressing search intent>
<200-300 words>

## H2: <next subtopic>
<200-300 words>

## FAQ
- <long-tail keyword question>?
- <long-tail keyword question>?

Internal links: <related pages>
```

### Landing Page Structure

- Benefit-driven headline
- Clear value proposition
- Social proof section
- FAQ for common objections
- Strong CTA

### Content Optimization (Existing Pages)

1. Audit existing page for missing keywords
2. Update title tag and meta description
3. Add missing subtopic sections
4. Refresh outdated information
5. Improve internal linking

### Search Console Opportunity Analysis

```
Rising Keywords (position 8-20, worth targeting):
| Keyword | Position | Impressions | CTR | Action |
|---------|----------|-------------|-----|--------|

Declining Keywords (lost 5+ positions):
| Keyword | Old Position | New Position | Action |
|---------|-------------|-------------|--------|
```

### Rules

- Write for humans first, search engines second
- Include the target keyword naturally in title, H1, first paragraph
- Use related keywords throughout (semantic SEO)
- Never keyword stuff or force unnatural phrasing
- Aim for 1000+ words for blog posts
- Never ignore search intent (match content to what users are looking for)
- Never copy competitor content structure exactly
- Always include meta descriptions (they drive CTR)
