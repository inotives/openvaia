## Ad Copywriting

> ~555 tokens

### Platform Character Limits

| Platform | Element | Max Length |
|----------|---------|-----------|
| Google Ads | Headline | 30 characters |
| Google Ads | Description | 90 characters |
| Meta/Facebook | Primary text | 125 characters (before "See more") |
| LinkedIn | Intro text | 150 characters (before truncation) |

### Psychological Angles

Label every variant with the angle used:

| Angle | Description | Example |
|-------|-------------|---------|
| Benefit-first | Lead with the outcome | "AI Code Reviews in Seconds" |
| Pain-point | Address the frustration | "Ship Bugs? Never Again" |
| Social proof | Numbers and credibility | "500K PRs Reviewed by AI" |
| Scarcity | Limited availability | "Only 50 Spots Left This Month" |
| Curiosity | Create information gap | "This App Replaced Our Most Stressful Decision" |

### Hook-Story-Offer Framework (Meta/Facebook)

1. **Hook:** Stop the scroll in the first line
2. **Story/Body:** Build context and value (2-3 sentences)
3. **CTA:** Clear action with low friction

### Ad Copy Generation Workflow

1. Confirm target audience, product, and campaign goal
2. Generate at least 3 variants per element
3. Label each variant with the psychological angle used
4. Include a clear CTA in every variant
5. Provide A/B test recommendation (which variants to test first)

### A/B Testing Strategy

- Test one variable at a time (headline, CTA, angle)
- Start with headline tests (highest impact on CTR)
- Run for statistical significance (minimum 100 conversions per variant)
- Document winning angles for future campaigns

### Funnel Stage Adaptation

| Stage | Goal | Tone |
|-------|------|------|
| Awareness | Introduce problem/solution | Educational, curiosity |
| Consideration | Differentiate from alternatives | Proof, comparison |
| Conversion | Drive action | Urgency, benefit, CTA |

### Rules

- Always provide at least 3 variants per element for A/B testing
- Include a clear CTA in every variant
- Never use clickbait, false urgency, or misleading claims
- Respect platform-specific character limits
- Label each variant with the psychological angle used
- Write copy that passes platform policy checks
